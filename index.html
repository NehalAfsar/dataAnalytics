Help & Instructions

Getting Started:

1. Upload your CSV or Excel file using drag & drop or file picker

2. Review your data in the preview section

3. Use filters to narrow down your data

Creating Charts:

1. Select chart type and axes in the configuration section

2. Click 'Add Chart' to create visualizations

3. Drag charts to rearrange them

Keyboard Shortcuts:

Ctrl+S: Save project | Ctrl+O: Load project | Ctrl+E: Export PNG
